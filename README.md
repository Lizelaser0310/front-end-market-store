## Features
- Product search
- Shopping cart
- Checkout proccess
- Product list with filters and ordering
- Credit card and Paypal payment methods
- Account creation and management
- User directions with geolocalization
- Order history
- Wish list
- Favorite products
- Category list
- Product packs by weeks
